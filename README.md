# Groupie Tracker

Groupie Tracker is a Go-based web application that tracks artists, their concert dates, and locations. It provides a clean web interface to view artist information and upcoming events.

---

## Features

- Fetch and display artist data via API calls.
- Display concert dates in a responsive grid layout.
- Highlight special dates in the calendar.
- Serve static assets (CSS) via `/assets/`.
- Proper error handling for failed API calls.
- Fully responsive layout using CSS Grid.

1. Clone the repository:


2. Run it

